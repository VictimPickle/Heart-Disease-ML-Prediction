# Visualization Images

This folder contains the generated visualization images from the analysis:

- `correlation_heatmap.jpg` - Feature correlation matrix
- `feature_importance.jpg` - Feature importance comparison between models
- `model_metrics.jpg` - Performance metrics comparison
- `false_negatives.jpg` - False negatives analysis
- `decision_tree.jpg` - Decision tree visualization

These images are generated automatically when you run `heart_disease_analysis.py`.
